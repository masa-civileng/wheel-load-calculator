# 小型輪荷重走行試験機 稼働計算

スマートフォンで使いやすい静的HTML5計算アプリです。サーバーやデータベースは不要で、計算はすべて端末内で行います。

## 主な機能

- 開始日時と走行回数から停止日時を計算
- 開始日時と停止日時から完了できる走行回数を逆算
- 1時間当たりの走行回数を変更可能（初期値 3,015回/時）
- 入力内容をブラウザ内に自動保存
- 計算結果をコピー
- PC・スマートフォンの両方に対応

## 計算条件

24時間連続稼働・途中停止なしを前提とします。停止日時からの逆算は、その時刻までに完了できる整数回数を切り捨てて表示します。
